Module 2: Matrix Algebra

Master the arithmetic of matrices: addition, multiplication, transposition, inverses, and factorizations.

4 Lessons
~3-4 hours
10 Practice Problems
Module Quiz

Learning Objectives

By the end of this module, you will be able to:

  • Perform matrix addition, scalar multiplication, and matrix multiplication
  • Apply properties of matrix arithmetic (associativity, distributivity, non-commutativity)
  • Compute the transpose of a matrix and use its properties
  • Find the inverse of a matrix using row reduction
  • Understand elementary matrices and their role in LU factorization
