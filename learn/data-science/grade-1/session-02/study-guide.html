Lesson Summaries

Lesson 1: Sorting by Size

Learn to arrange objects from smallest to biggest and sort them into size groups.

Lesson 2: Sorting by Shape

Circles, squares, triangles — sort objects by their shape. What shapes do you see?

Lesson 3: Sorting by Texture

Smooth, rough, soft, hard — touch your objects and sort them by how they feel.

Lesson 4: Same Objects, Different Groups

Sort the SAME collection three different ways. See how the groups change each time!

Review Questions

  1. How many ways can you sort the same group of objects?
  2. Which of these is sorting by SIZE?
  3. What does 'texture' mean?
  4. If you sort toy animals first by color, then by size, will the groups be the same?
  5. Which is a shape you could sort by?
  6. Why is it useful to sort the same things different ways?
  7. What is an 'attribute'?
  8. If you sort blocks by shape and get 10 cubes, 5 cylinders, and 3 pyramids, which shape group is smallest?
