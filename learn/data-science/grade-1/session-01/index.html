Session 1: I Can Sort!

Sort a collection of objects by color. Discover that putting things in groups helps you see patterns.

4 Lessons
~2 hours total
6 Practice Activities
Session Quiz

Lessons 1-4
Practice
Session Quiz
Review

Learning Objectives

By the end of this session, you will be able to:

  • Identifying colors in a collection of objects
  • Sorting objects into groups by one attribute (color)
  • Counting objects in each group
  • Comparing groups: most, fewest, same

Session Lessons

1

Looking at Colors

Learn to identify and name colors in a collection. Every object has a color — and color is your first sorting tool.

~30 minutes Discussion

Start Lesson 1
2

Sorting Into Piles

Take a mixed collection and sort it by color into separate piles. See how groups form naturally.

~30 minutes Activity

Start Lesson 2
3

Counting Each Pile

Count how many objects are in each color group. Practice saying 'There are ___ red ones.'

~30 minutes Activity

Start Lesson 3
4

Which Group Is Biggest?

Compare your piles. Which color has the most? Which has the fewest? How do you know?

~30 minutes Discussion

Start Lesson 4
