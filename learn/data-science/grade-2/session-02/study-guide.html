Lesson Summaries

Lesson 1: What Is Measuring?

Measuring means using a tool to find out exactly how much. It is more precise than guessing!

Lesson 2: Using a Ruler

Learn to use a ruler correctly: start at zero, read the number at the end. Measure in inches or centimeters.

Lesson 3: Measuring 10 Objects

Measure 10 different objects with your ruler. Record each measurement in a table with the object name and number.

Lesson 4: Units and Labels

Always write the unit! '7 inches' means something. '7' alone does not. Learn why units matter.
