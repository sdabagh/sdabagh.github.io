Check Your Understanding

1. What is an increasing trend?

Answer: When values generally go up over time. Not every point needs to be higher than the last — the overall direction is what matters.

2. What is a stable trend?

Answer: When values stay approximately the same over time. There may be small fluctuations, but the overall level does not change much.

3. Can a trend change direction?

Answer: Yes! Data might increase for a while, then decrease, then stabilize. Identifying where trends change is an important skill.

4. How do you describe a trend in words?

Answer: State the time period, the direction, and the approximate amount. Example: 'Between January and June, daily temperatures increased from about 40F to 80F.'

Key Takeaways

  • Increasing trend: values generally going up over time
  • Decreasing trend: values generally going down over time
  • Stable trend: values staying about the same
