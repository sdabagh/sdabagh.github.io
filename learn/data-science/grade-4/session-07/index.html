Learning Objectives

By the end of this session, you will be able to:

  • Navigate a spreadsheet: rows, columns, cells
  • Enter data into a spreadsheet and format it
  • Use basic formulas: SUM, AVERAGE, COUNT, MAX, MIN
  • Create a chart from spreadsheet data with one click

Why This Matters

Spreadsheets are the most widely used data tool in the world. Learning the basics now gives you a skill you will use for the rest of your life.
