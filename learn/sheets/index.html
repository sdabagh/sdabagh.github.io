📊 Excel & Google Sheets for Data

Work with data using the world's most widely-used tools. No downloads needed — everything runs in Google Sheets (free).

🔹 8 Modules 🔹 Beginner Friendly 🔹 100% Free 🔹 Google Sheets & Excel 🔹 No Downloads Required
📌

Prerequisites

No experience needed. Basic computer skills (mouse, keyboard) is all you need. If you can browse a website, you can take this course.

✅

Works with Both Google Sheets and Excel

All examples are shown in Google Sheets — completely free with a Google account, no installation required. Key differences from Excel are noted throughout. If you have Excel, everything in this course works there too.

What You'll Learn

📋
Navigate spreadsheets confidently Cells, rows, columns, formulas, formatting — the full interface.
➕
Write powerful formulas SUM, AVERAGE, IF, VLOOKUP, XLOOKUP, and more.
📊
Summarize data with pivot tables Turn 500 rows into a clear summary in seconds.
📈
Create charts and visualizations Bar, line, pie, and scatter — and when to use each.
🧹
Clean messy data TRIM, PROPER, CLEAN, handling blanks and duplicates.
🏆
Complete a full data project Import, clean, analyze, visualize, and report — end to end.
