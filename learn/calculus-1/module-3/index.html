Module 3: Differentiation Rules

Now that you understand what a derivative is, it is time to build your toolkit. In this module you will learn the product, quotient, and chain rules, differentiate trig functions, and tackle implicit differentiation.

Learning Objectives

By the end of this module, you will be able to:

  • Apply the Product Rule and Quotient Rule to differentiate products and quotients of functions
  • Use the Chain Rule to differentiate composite functions
  • Differentiate all six trigonometric functions
  • Perform implicit differentiation to find dy/dx for implicitly defined curves
  • Combine multiple rules in a single differentiation problem
