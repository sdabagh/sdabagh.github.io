Module 6: Integration

Welcome to the second major branch of calculus! This module introduces antiderivatives, Riemann sums, the definite integral, and the Fundamental Theorem of Calculus that connects differentiation and integration.

Learning Objectives

By the end of this module, you will be able to:

  • Compute antiderivatives using basic rules
  • Approximate areas under curves using Riemann sums
  • Evaluate definite integrals using the Fundamental Theorem of Calculus
  • Apply FTC Part 2 and the Net Change Theorem
