Learning Objectives

By the end of this module, you will be able to:

  • Apply the First Derivative Test to classify critical points
  • Use the Second Derivative Test and determine concavity
  • Sketch curves using derivatives, intercepts, and asymptotes
  • Solve optimization problems from real-world contexts
