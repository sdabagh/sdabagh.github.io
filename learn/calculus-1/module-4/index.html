Module 4: Applications of Derivatives I

Put your differentiation skills to work! This module covers real-world applications including related rates, linear approximation, finding extreme values, and the powerful Mean Value Theorem.

Learning Objectives

By the end of this module, you will be able to:

  • Solve related rates problems by differentiating with respect to time
  • Use linearization and differentials to approximate function values
  • Find absolute and local extreme values using critical points
  • State and apply the Mean Value Theorem and Rolle's Theorem
