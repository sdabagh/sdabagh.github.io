Module 2: The Unit Circle & Right Triangle Trigonometry

Define the trig functions using the unit circle, evaluate them at special angles, and solve right triangles.

4 Lessons
~3-4 hours
10 Practice Problems
Module Quiz

Learning Objectives

By the end of this module, you will be able to:

  • Define sine and cosine using the unit circle
  • Evaluate trig functions at special angles (0, π/6, π/4, π/3, π/2) and their multiples
  • Apply right triangle definitions (SOH-CAH-TOA) to find sides and angles
  • Solve right triangles and real-world application problems
