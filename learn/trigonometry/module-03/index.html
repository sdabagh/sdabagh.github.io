Module 3: Trig Functions of Any Angle

Extend trig functions to all angles using reference angles, and master the fundamental trig identities.

4 Lessons
~3 hours
10 Practice Problems
Module Quiz

Learning Objectives

By the end of this module, you will be able to:

  • Evaluate trig functions of any angle using reference angles and the coordinate definition
  • Find all six trig functions given one value and the quadrant
  • Apply reciprocal, quotient, and Pythagorean identities
  • Use even-odd properties and cofunction identities to simplify expressions
