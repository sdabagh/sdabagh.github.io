Data Analyst: Designed with Neurodivergence in Mind

A free, structured path from foundations to job-ready — built for the way YOUR brain works

14 Modules 6 Phases ~14 Weeks 100% Free Free Tools Only Self-Paced
🧠

Designed for neurodivergent learners — and everyone who learns better with structure

This course was built with ADHD, autism, dyslexia, and anxiety in mind. Every module has the same predictable 7-part format so there are never any surprises. Short bursts only — nothing over 25 minutes. The WHY always comes before the HOW. Mandatory brain breaks built in. Color-coded sections so you always know where you are. You are always in control. Go at your own pace. Stop anytime. Come back anytime.

📅 Timeline

This path is designed for approximately 14 weeks at a pace of one module per week. But there is no schedule. Go faster if you want. Go slower if you need to. Take a month off. Come back. The course will be here.

Phases 1–3 build on each other — we recommend doing them in order. Phases 4–6 can overlap. SAP (Phase 5) can be started any time after Phase 1.
