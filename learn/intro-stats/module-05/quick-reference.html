z-table Usage Guide

Standard z-table gives:

  • Area to the LEFT of z
  • This equals P(Z < z)
  • For z = 1.5 ‚Üí 0.9332 means 93.32% below

Reading the table:

  • Row = first two digits (e.g., 1.2)
  • Column = second decimal (e.g., 0.05)
  • Example: z = 1.25 ‚Üí row 1.2, col 0.05

Decision Tree

Problem Type?

  • Given x, find P? ‚Üí x to z, use table
  • Given P, find x? ‚Üí table to z, z to x
  • "Less than"? ‚Üí Direct table lookup
  • "Greater than"? ‚Üí 1 ‚àí table value
  • "Between"? ‚Üí Subtract two table values
  • Percentile? ‚Üí Inverse lookup
