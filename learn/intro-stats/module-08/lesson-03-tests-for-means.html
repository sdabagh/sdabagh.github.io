P-Value Approach

üìñ Definition: P-Value

The p-value is the probability of obtaining a test statistic as extreme as (or more extreme than) the one calculated, assuming the null hypothesis is true.

Decision Rule:

  • If p-value ‚â§ Œ± ‚Üí Reject H‚ÇÄ
  • If p-value > Œ± ‚Üí Fail to reject H‚ÇÄ
Interpreting P-Values:
  • p < 0.01: Very strong evidence against H‚ÇÄ
  • 0.01 ‚â§ p < 0.05: Strong evidence against H‚ÇÄ
  • 0.05 ‚â§ p < 0.10: Weak evidence against H‚ÇÄ

3. T-Test for Population Mean

When to Use

Use the t-test when:

  • You're testing a claim about a population mean (Œº)
  • The population standard deviation (œÉ) is unknown
  • Either n ‚â• 30 OR the population is normally distributed

Test Statistic for T-Test

t = (xÃÑ - Œº‚ÇÄ) / (s / ‚àön)

Where:
xÃÑ = sample mean
Œº‚ÇÄ = hypothesized population mean (from H‚ÇÄ)
s = sample standard deviation
n = sample size
Degrees of freedom: df = n - 1

Key Difference from Z-Test:
  • We use sample standard deviation s instead of œÉ
  • We use the t-distribution instead of standard normal distribution
  • Critical values depend on degrees of freedom (df = n - 1)
  • The t-distribution has heavier tails than normal distribution (more conservative)

Finding Critical Values for T-Test

Critical values come from the t-distribution table using:

  • Degrees of freedom: df = n - 1
  • Significance level: Œ±
  • Test type: one-tailed or two-tailed

  • H‚ÇÄ: Œº = 10 (weight loss is 10 pounds as claimed)

  • Œ± = 0.05

t = (8.2 - 10) / (3.5 / ‚àö25)

Left-tailed test, Œ± = 0.05, df = 24 ‚Üí critical value = -1.711 (from t-table)

Step 4: Make decision

t = -2.57 < -1.711, so t is in the critical region

Decision: Reject H‚ÇÄ

Step 5: Conclusion

There is sufficient evidence at the 0.05 significance level that the average weight loss is less than the claimed 10 pounds. The program appears to be less effective than advertised.

Reasoning: In real-world scenarios, we almost never know the true population standard deviation (œÉ) for salaries. We would only have the sample standard deviation (s) from our study, so we must use the t-test.
