üéØ Learning Objectives

By the end of this module, you will be able to:

  • Explain what a confidence interval is and what it represents
  • Distinguish between point estimates and interval estimates
  • Interpret confidence levels (90%, 95%, 99%) correctly
  • Construct confidence intervals for population means using the t-distribution
  • Construct confidence intervals for population proportions
  • Determine when to use z vs t distributions for confidence intervals
  • Calculate margin of error and understand factors that affect it
  • Determine appropriate sample sizes for desired precision
  • Interpret confidence intervals correctly (common misconceptions)
