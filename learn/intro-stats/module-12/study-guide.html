1. Overview of Chi-Square Tests

Purpose: Chi-square tests analyze categorical data to determine if observed frequencies differ significantly from expected frequencies.

Three Types of Chi-Square Tests

  • Goodness of Fit: Tests if sample data fits an expected distribution
  • Independence: Tests if two categorical variables are related
  • Homogeneity: Tests if multiple populations have the same distribution

When to Use Chi-Square

  • ‚úì You have categorical data (counts/frequencies)
  • ‚úì You want to compare observed vs. expected frequencies
  • ‚úì You need to test relationships between categorical variables
  • ‚úó Do NOT use for quantitative data (use t-tests, ANOVA instead)

2. Chi-Square Distribution

Properties

  • Always positive: œá¬≤ ‚â• 0 (cannot be negative)
  • Right-skewed: Tail extends to the right
  • Shape depends on df: More degrees of freedom ‚Üí more symmetric
  • Mean = df: The expected value equals the degrees of freedom

Key Characteristics

  • Unlike the normal distribution, chi-square is NOT symmetric (except with very large df)
  • We always use the RIGHT tail for rejection region
  • Different df values create different distribution curves

4. Chi-Square Test of Independence

When to Use

  • TWO categorical variables
  • ONE sample (classified by both variables)
  • Want to test if variables are related

Step-by-Step Procedure

  1. State hypotheses
    • H‚ÇÄ: The two variables are independent (no association)

  2. Check conditions (same as goodness of fit)

  5. Find degrees of freedom

  6. Find p-value or critical value

5. Chi-Square Test of Homogeneity

When to Use

  • ONE categorical variable
  • MULTIPLE samples/populations
  • Want to test if distributions are the same across groups

Step-by-Step Procedure

  1. State hypotheses
    • H‚ÇÄ: The distribution is the same for all populations
    • H‚Çê: At least one population has a different distribution
  2. Check conditions (same as others)

  5. Find degrees of freedom

  6. Find p-value or critical value

Key Difference from Independence

Calculations are IDENTICAL, but study design differs:

  • Independence: One sample, two variables
  • Homogeneity: Multiple samples, one variable

7. Choosing the Right Test

Decision Process

  1. Is the data categorical?
    • No ‚Üí Use t-test, ANOVA, or regression (NOT chi-square)
    • Yes ‚Üí Continue
  2. How many categorical variables?
    • One variable ‚Üí Goodness of Fit OR Homogeneity (check #3)
    • Two variables ‚Üí Independence OR Homogeneity (check #3)
  3. How many samples?
    • One sample ‚Üí Goodness of Fit (if 1 variable) or Independence (if 2 variables)
    • Multiple samples ‚Üí Homogeneity

Quick Decision Rules

  • Testing if data matches a theory/model? ‚Üí Goodness of Fit
  • One sample classified by TWO variables? ‚Üí Independence
  • Comparing MULTIPLE groups on ONE variable? ‚Üí Homogeneity
  • Random assignment to treatment groups? ‚Üí Usually Homogeneity

8. Conditions and Assumptions

‚ö†Ô∏è Required Conditions (ALL Tests)

  1. Random Sampling or Random Assignment
    • Ensures representative data
    • If violated: Cannot generalize to population
  2. Independence of Observations
    • Each observation must be independent
    • If violated: p-values are unreliable
  3. All Expected Frequencies ‚â• 5
    • MOST CRITICAL CONDITION
    • Check EVERY expected cell
    • If violated: Combine categories, collect more data, or use Fisher's exact test (for 2√ó2)
  4. Categorical Data (Counts)
    • Must use frequencies, NOT proportions or percentages

10. Common Mistakes to Avoid

‚ùå Top 10 Mistakes

  1. Using proportions instead of counts
    • WRONG: Entering 0.35 (35%)
    • RIGHT: Entering 35 (the count)
  2. Not checking expected frequency condition
    • Always calculate expected values BEFORE computing œá¬≤
    • Check that ALL are ‚â• 5
  3. Confusing independence and homogeneity
    • Look at study design, not just the table
  4. Using wrong degrees of freedom
    • Goodness of fit: k - 1
    • Independence/Homogeneity: (r-1)(c-1)
  5. Trying to use one-tailed test
    • Chi-square is ALWAYS right-tailed
  6. Using chi-square for quantitative data
    • Chi-square is for categories only!
  7. Claiming causation from association
    • Association does not imply causation
  8. Forgetting to sum over ALL cells
    • œá¬≤ must include contribution from every cell
  9. Not stating conclusion in context
    • Always relate back to the original problem
  10. Rounding too early
    • Keep extra decimals in calculations
    • Round only final answer
