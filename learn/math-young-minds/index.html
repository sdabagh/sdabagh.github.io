K–5 Math Curriculum

Math for Young Minds

A neurodivergent-friendly elementary math curriculum, designed for short attention spans and real-world learning. Free forever. Common Core anchored.

The big idea

Math is something you do with the world, not to a worksheet. Every session starts with a real object a child can touch — cars, cookies, crayons — and only later moves to symbols on a page. Sessions are 15–25 minutes (attention-span-aware), use the same predictable 5-block structure every time, and have no timed tests.

What's in each session

📋 Session plan

A scripted, time-marked plan for the teacher — pick it up cold and run.

✏️ Worksheet

One page for the student — visual schedule, big drawing space, scaffolded practice.

🏠 Family guide

A take-home note for parents with a 1-minute home activity.

🔑 Cheat sheet

A visual one-page reference of the core idea.

Designed for every kind of mind
