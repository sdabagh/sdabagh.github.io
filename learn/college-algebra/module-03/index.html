Learning Objectives

By the end of this module, you will be able to:

  • Identify key features of polynomial functions including degree, leading coefficient, and end behavior
  • Find zeros of polynomial functions and understand multiplicity
  • Factor polynomials using multiple methods including the Rational Root Theorem
  • Analyze and graph rational functions including identifying asymptotes and holes
  • Solve polynomial and rational inequalities using test intervals and sign charts
