Post-Assessment

Module 4: Exponential & Logarithmic Functions

About This Assessment

This post-assessment measures how much you've learned about exponential and logarithmic functions after completing the module. These questions are similar in difficulty to the pre-assessment, allowing us to measure your learning gains.

Time: 5-10 minutes | Questions: 5 | Purpose: Measure learning gains

Question 1

Evaluate g(x) = 3x when x = 4.

Question 2

Which function represents exponential decay?

Question 3

Evaluate log5(25).

Question 4

If you invest $2000 at 4% annual interest compounded annually, which formula represents the amount A after t years?

Question 5

A radioactive substance decays so that half of its mass remains after each year. Which type of function best models this situation?
