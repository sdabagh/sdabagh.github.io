5 Identify Functions Easy

Determine whether each relation is a function. Explain why or why not.

  1. {(1, 2), (3, 4), (5, 6)}
  2. {(2, 3), (3, 4), (2, 5)}
  3. {(0, 1), (1, 1), (2, 1)}

Solution:

(a) {(1, 2), (3, 4), (5, 6)}: ‚úÖ YES, this IS a function

Reasoning: Each input (x-value) appears only once: 1‚Üí2, 3‚Üí4, 5‚Üí6. Every input has exactly ONE output, which satisfies the definition of a function.

(b) {(2, 3), (3, 4), (2, 5)}: ‚ùå NO, this is NOT a function

Reasoning: The input x = 2 appears twice with different outputs: 2‚Üí3 and 2‚Üí5. Since the input 2 maps to TWO different outputs, this violates the function rule. A function requires each input to have exactly ONE output.

(c) {(0, 1), (1, 1), (2, 1)}: ‚úÖ YES, this IS a function

Reasoning: Each input appears only once: 0‚Üí1, 1‚Üí1, 2‚Üí1. Even though all three inputs map to the SAME output (1), that's perfectly fine! A function can have multiple inputs that produce the same output. What matters is that each input has only ONE output.

Function Rule: Each input (x) must map to EXACTLY ONE output (y). Multiple x's can share the same y, but one x cannot have multiple y's!

6 Vertical Line Test Medium

For each description, determine if the graph represents a function using the vertical line test:

  1. A parabola opening upward (like y = x¬≤)
  2. A circle
  3. A straight line with slope 2
  4. A sideways parabola (like x = y¬≤)

Solution:

(a) Parabola opening upward: ‚úÖ IS a function

Vertical Line Test: If you draw any vertical line (like x = 2), it will only intersect the parabola at ONE point. Since no vertical line hits the graph more than once, it passes the vertical line test and IS a function.

(b) Circle: ‚ùå NOT a function

Vertical Line Test: If you draw a vertical line through the middle of a circle, it will intersect the circle at TWO points (top and bottom). Since at least one vertical line hits the graph more than once, it fails the vertical line test and is NOT a function.

(c) Straight line with slope 2: ‚úÖ IS a function

Vertical Line Test: A non-vertical straight line (like y = 2x + 1) will only be intersected by any vertical line at ONE point. It passes the test and IS a function. (Note: A VERTICAL line like x = 3 would NOT be a function!)

(d) Sideways parabola: ‚ùå NOT a function

Vertical Line Test: For a sideways parabola like x = y¬≤, a vertical line will intersect the graph at TWO points (for example, at x = 4, you get y = 2 and y = -2). It fails the test and is NOT a function.

Remember: Vertical line test = If ANY vertical line hits the graph MORE THAN ONCE, it's NOT a function!

10 Division with Domain Restrictions Hard
