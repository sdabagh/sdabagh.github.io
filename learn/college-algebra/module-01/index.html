Learning Objectives

By the end of this module, you will be able to:

  • Classify real numbers and plot points on the Cartesian coordinate plane
  • Calculate distance and midpoint between two points using formulas
  • Distinguish between relations and functions using multiple representations
  • Evaluate functions using function notation and determine domain and range
  • Perform function arithmetic (addition, subtraction, multiplication, division)
  • Graph parent functions and apply transformations (shifts, stretches, reflections)

Why This Matters

Functions are the language of mathematics and science! Every formula you'll use in calculus, physics, economics, and computer science is built on the foundation of functions.

This module gives you the essential skills to:

  • Understand relationships between variables in science and economics
  • Model real-world situations mathematically
  • Succeed in calculus and higher-level STEM courses
  • Develop problem-solving skills valued in all careers
  • Build confidence in your mathematical abilities

Tips for Success

  • Practice, practice, practice - Algebra is a skill that improves with repetition. Do the problems!
  • Master the basics first - Understanding parent functions makes transformations much easier.
  • Draw graphs - Visualizing functions helps you understand them deeply.
  • Watch for tricky signs - Horizontal shifts are opposite of the sign: f(x-3) shifts RIGHT!
  • Check your domain - Always consider restrictions like division by zero and square roots of negatives.
  • Don't fear mistakes - Every mathematician makes errors. Learn from them and keep going!
