Mean / Average

How to Calculate the Mean

The mean is the average of all values.

Steps:

  1. Add up all the values
  2. Count how many values you have
  3. Divide the sum by the count

Example:

Data: 4, 7, 9, 12, 18

Sum = 4 + 7 + 9 + 12 + 18 = 50
Count = 5
Mean = 50 ÷ 5 = 10

Tip: The mean can be affected by outliers (very high or low values)!

Median

How to Find the Median

The median is the middle value when data is sorted.

Steps:

  1. Sort data from smallest to largest
  2. If odd count: pick the middle number
  3. If even count: average the two middle numbers

Example (odd count):

Data: 3, 7, 8, 12, 15

Already sorted
Count = 5 (odd)
Middle = 3rd value = 8

Tip: Median is resistant to outliers - it represents the "typical" value better than mean when outliers exist!

Mode

How to Find the Mode

The mode is the value that appears most frequently.

Steps:

  1. Count how often each value appears
  2. Find the value(s) with the highest frequency
  3. If all values appear equally, there is no mode

Example:

Data: 2, 3, 3, 5, 5, 5, 7, 9

2 appears 1 time
3 appears 2 times
5 appears 3 times ← Most frequent
7, 9 appear 1 time each

Mode = 5

Tip: A dataset can have no mode, one mode, or multiple modes!

Standard Deviation

Standard Deviation

Standard deviation measures how spread out data is from the mean.

Steps:

  1. Find the mean
  2. Subtract mean from each value (deviations)
  3. Square each deviation
  4. Average the squared deviations (variance)
  5. Take the square root

Example:

Data: 2, 4, 6, 8, 10

Mean = 6
Deviations: -4, -2, 0, 2, 4
Squared: 16, 4, 0, 4, 16
Variance = (16+4+0+4+16)/5 = 8
SD = √8 = 2.83

Tip: Larger SD = more spread out data!

Variance

Variance

Variance measures the average squared distance from the mean.

Steps:

  1. Find the mean
  2. Subtract mean from each value
  3. Square each deviation
  4. Average all squared deviations

Example:

Data: 1, 3, 5, 7, 9

Mean = 5
Deviations: -4, -2, 0, 2, 4
Squared: 16, 4, 0, 4, 16
Variance = (16+4+0+4+16)/5 = 8

Tip: Variance is in squared units. Standard deviation (√variance) is in original units!

Confidence Intervals

Confidence Intervals

A confidence interval gives a range where the population parameter likely falls.

95% CI Formula:

CI = x̄ ± (z* × SE)

Where:
• x̄ = sample mean
• z* = critical value (1.96 for 95%)
• SE = s/√n (standard error)

Example:

n=36, x̄=78, s=12

SE = 12/√36 = 2
ME = 1.96 × 2 = 3.92
CI = 78 ± 3.92 = (74.08, 81.92)

Interpretation: We are 95% confident the true population mean is between 74.08 and 81.92.

Hypothesis Testing

Hypothesis Testing

Hypothesis testing determines if sample data supports a claim about a population.

Steps:

  1. State H₀ (null hypothesis) and Hₐ (alternative)
  2. Choose significance level (α, usually 0.05)
  3. Calculate test statistic
  4. Find critical value or p-value
  5. Make decision: Reject H₀ or Fail to reject

Example:

Test if μ > 65 (α=0.05)
Data: n=25, x̄=67, s=4

H₀: μ = 65
Hₐ: μ > 65
t = (67-65)/(4/√25) = 2.5
Critical value: t* = 1.711
2.5 > 1.711 → Reject H₀

Conclusion: Evidence supports μ > 65

Probability

Probability

Probability measures the chance of an event occurring.

Formula:

P(event) = (Favorable outcomes) / (Total possible outcomes)

Example:

Rolling two dice, what is P(sum = 7)?

Total outcomes: 36
Favorable outcomes for sum=7:
(1,6), (2,5), (3,4), (4,3), (5,2), (6,1) = 6

P(sum=7) = 6/36 = 1/6 = 0.167 or 16.7%

Key Properties:
• 0 ≤ P ≤ 1
• P = 0 means impossible
• P = 1 means certain

Correlation

Correlation

Correlation measures the strength and direction of a linear relationship between two variables.

Correlation coefficient (r):
• Range: -1 to +1
• r near +1 = strong positive relationship
• r near -1 = strong negative relationship
• r near 0 = weak/no relationship

Example:

r = 0.85

Interpretation:
• 0.85 is close to +1
• Strong positive correlation
• As X increases, Y tends to increase
• r² = 0.72 (72% of variation explained)

Important: Correlation ≠ Causation! Just because two variables are correlated doesn't mean one causes the other.

Sampling

Sampling

Sampling is selecting a subset of a population to study.

Types of Sampling:

  • Random: Every member has equal chance
  • Stratified: Divide into groups, sample from each
  • Cluster: Randomly select groups, survey all in selected groups
  • Systematic: Every nth member

Example:

Population: 1,000 students
Sample: 100 randomly selected

Why sample?
• Too expensive/time-consuming to survey all
• Random sample is representative
• Can infer population characteristics

Key: Random selection prevents bias!
