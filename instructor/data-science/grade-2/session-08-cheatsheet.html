Session 8 — Our Class Data Book (Capstone)

Instructor Cheatsheet · Data Science for Young Minds · Grade 2 · Ages 7–8
~50 min Ages 7–8 Session 8 of 8 ND-Friendly Capstone Celebration
Session Goal

Students complete their bar graph using Session 7 tally chart data, write one conclusion sentence, assemble the Class Data Book, and celebrate becoming data scientists. Every student's page goes in the book — no one is left out.

Bar Graph Drawing Guide (Post This on Board)
1Look at your tally chart from Session 7. Find the totals for each answer choice.
2Draw two lines: one going up (y-axis) and one going across (x-axis).
3Write numbers on the y-axis going up: 0, 1, 2, 3, 4, 5, 6, 7, 8.
4Write your answer choices along the bottom (x-axis).
5For each answer, count its total and draw a bar that tall. Use a ruler!
6Color each bar a different color.
7Write a title at the top of your graph.
8Check: does each bar height match the number in your tally chart?
The worksheet has a pre-drawn grid — students only need to draw bars, label axes, and add a title. Reduces frustration significantly.
Sentence Frame Poster (Write on Board)
"My question was ___ and I found out that ___."
"The most popular answer was ___ with ___ votes."
"I was surprised that ___ because ___."
"Next time I would ask ___."
Students pick ONE sentence frame. Drawing a picture conclusion is also accepted — see ND tips.
Brain Break (min 44) — Data Dance
  • Call out a data word from the vocab list
  • Students make up a move for that word
  • E.g., "TALLY!" → students make a slashing motion with arm
  • "BAR GRAPH!" → students stretch arms up like a tall bar
  • "SURVEY!" → students pretend to ask a question to their neighbor
  • Repeat 4–5 words, keep it fast and fun

Assembling the Class Data Book

📌 Collect all student bar graph pages
📌 Add cover page (students can decorate theirs)
📌 Bind with staples, binder rings, or a folder
📌 Place it where the class can read it
📌 Every student's page is included — no exceptions
📌 If a student didn't finish, include a partial page
📌 Read 2–3 pages aloud together as a class
📌 Let students point to their own page proudly
📌 Optional: share with another class or parents

Non-Verbal + Alternate Completion Options

  • Student draws bars without labeling — that counts
  • Student colors a pre-labeled bar graph teacher prepares
  • Student dictates conclusion sentence to teacher/aide
  • Student draws a picture instead of writing conclusion
  • Cover page can be illustrated only — no words required

ND-Friendly Instructor Notes

  • Sentence frame always visible — on board + on worksheet. Never remove it until every student has written their sentence.
  • Pre-drawn graph grid — the worksheet grid is already ruled so students only draw bars. Removes axis-drawing as a barrier.
  • Honor drawing as conclusion — a student who draws their finding instead of writing it has still demonstrated data understanding. Accept it fully.
  • Every page goes in the book — partial, messy, or minimally completed pages are included without exception. Belonging matters more than perfection.
  • Celebrate every contribution — at assembly, say each student's name as you add their page. This 5 seconds of recognition is significant.
  • Closure ritual — end with "We are data scientists!" as a class call-and-response. Simple, memorable, identity-affirming.
